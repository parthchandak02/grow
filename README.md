# Grow — a generative garden

Click to plant seeds. Watch them grow into unique botanical forms.

A single-file generative art project using native Canvas 2D API. Zero dependencies, works offline, open in any browser.

🌱 [Live Demo](https://parthchandak02.github.io/grow/)

## How it works

- Tap/click anywhere to plant a seed
- Each seed grows into a unique plant over ~10 seconds
- Branches split organically, leaves appear in sage and terracotta tones
- Fully grown plants sway gently in a breeze
- Up to 30 plants — the oldest fades out when new ones are planted

## Design

- Cream paper background with dark ink branches
- Leaf tones: sage (#7a9b7a) and terracotta (#c46a4d)
- Vintage botanical illustration aesthetic
- Built with native Canvas 2D API — no dependencies, works offline

## Single-file pattern

This project follows a reusable pattern for creative tech projects:
- Zero dependencies (no p5.js, no Three.js, no CDN)
- Works from `file://` — open directly in any browser
- Mobile-friendly with touch support
- GitHub Pages ready — push `index.html` to `main`, enable Pages, done
